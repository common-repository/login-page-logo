This Plugin lets you customize the Logo of the admin login page panel. Helpful if you need to add a client's (or your own) logo.

== Description ==
Customize your admin login page: change the default logo and add your own or your client's logo.
You also can add width and height to your login page logo.

== Installation ==

1. Download the zip. 
2. Go to Admin Panel -> Plugins -> Add new.
3. Search, install and activate the plugin.

== OR ==

1. Download the zip. 
2. Go to Admin Panel -> Plugins -> Add new.
3. Search, install and activate the plugin.
4. Click "Upload" and upload the .zip file (you can also upload the unzipped folder to your "/wp-content/plugins/" directory via FTP)
5. Activate the plugin through the 'Plugins' menu in WordPress
6. The Plugin settings appear at Settings -> Login Page Logo.
